# 🕰️ 타임 트래블러: 유물 원정대 (Time Traveler Quest)

![Badge](https://img.shields.io/badge/Made_by-Dad-blue) ![Pythonista](https://img.shields.io/badge/Platform-Pythonista_3-yellow) ![Level](https://img.shields.io/badge/Level-Elementary_School-green)

**아빠가 직접 만든 초등학생 맞춤형 역사 퀴즈 게임!** 과거로 시간 여행을 떠나 역사 퀴즈를 풀고, 잃어버린 9개의 전설적인 유물들을 수집해 보세요!

## ✨ 게임 특징 (Features)
* 🗺️ **8개의 역사 스테이지**: 선사시대 매머드 사냥꾼부터 STAGE 8 조선의 독립운동가들까지!
* 💯 **10문제 챌린지**: 각 스테이지마다 10문제 중 7문제를 맞혀야 유물을 획득할 수 있습니다.
* 🎒 **나만의 보물 도감**: 내가 모은 유물들을 팝업 도감에서 한눈에 확인할 수 있습니다.
* 🤖 **AI 비밀 미션**: STAGE를 다 깼다면? 인공지능(Gemini)이 즉석에서 만들어주는 무한 랜덤 미션에 도전하세요!
* 💾 **자동 저장**: 게임을 껐다 켜도 내가 모은 유물과 깬 스테이지는 그대로 남아있습니다.

## 📱 아이패드에서 실행하는 방법 (How to Play)
이 게임은 아이패드의 **Pythonista 3** 앱에서 실행됩니다.

1. 아이패드에서 이 GitHub 페이지에 들어옵니다.
2. `time_travel_quest.py` 파일을 누르고 **'Raw'** 버튼을 눌러 전체 코드를 복사합니다.
3. 아이패드의 **Pythonista** 앱을 열고 새 파일(Empty Script)을 만들어 코드를 붙여넣습니다.
4. **[중요]** 코드 16번째 줄에 있는 `self.API_KEY` 부분에 아빠가 알려준 **비밀 API 키**를 따옴표 안에 입력합니다.
